# GimmeBuffsPls

GimmeBuffsPls is a World of Warcraft addon to check which buffs user has and ask for the missing ones.

## Commands

* `buff me` - ask for buffs in party chat
* `buff party` - write to party chat which teamates miss which buffs
* `buff check` - print into chat window which buffs teamates are missing (only user can see this message)
* `buff check me` - print into chat window which buffs user is missing (only user can see this message)
